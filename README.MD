# USB to Sega Mini
A Controller adapter for the Sega Saturn, MegaDrive and Genesis!

***Built on [Sonik-br's](https://github.com/sonik-br) USB2RetroConsole firmware***
Link to the firmware ***[HERE!](https://github.com/sonik-br/usb2retroconsole-fw)***

![Project Image](/Images/PCB.png)

## About this project
This github repo includes the source KICAD files for making modifications, and has it's GERBER, BOM and CPL in the releases tab. You may need to make modifications to find suitable parts
I have manually excluded the DB9 connector and Saturn connector as to freely allow installation of whichever you choose.


## Features:
<ul>
	<li>Support for various wired controllers (Original Xbox, 360, One, Series, PS3, PS4, PS5, Switch and any XINPUT capable device</li>
	<li>Low latency conversion, under or around 1 frame</li>
	<li>Emulation of various controllers (Model 2, 3D Pad, Arcade Racer/Steering Wheel, Mission Stick (3 axis mode), Shuttle mouse)</li>
	<li>Plug and play! No console modification required!</li>
</ul>
